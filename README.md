# Calcified Rock Despawn Timer

Shows an estimate of the remaining time before a calcified rock despawns.

![demo.png](demo.png)
## Features

- Timer display modes: pie timer, tick counter, or minutes:seconds.
- Despawn timer for when a calcified rock starts being mined
- Adjusts despawn timer when a waterstream appears on a calcified rock
- Reset timer for when a previously mined rock will reset its despawn timer (when there is no longer any activity on said rock)
- UI customization for colors and size.
